Gizlilik Politikası

Bu gizlilik politikası, HızlıSoft Caller ID uygulamasının kullanıcı verilerini nasıl topladığını, kullandığını, sakladığını ve koruduğunu açıklar...

Bu gizlilik politikası, HızlıSoft Caller ID uygulamasının kullanıcı verilerini nasıl topladığını, kullandığını, sakladığını ve koruduğunu açıklar. Uygulamayı kullanarak bu politikada belirtilen şartları kabul etmiş olursunuz. 1. Toplanan Bilgiler Uygulamamız aşağıdaki bilgileri toplar ve işler: Rehber (Kişiler) Bilgileri: Arayan kişinin ismini görüntülemek için cihaz rehberine erişim sağlanır. Arama Kayıtları ve Arama Durumu: Gelen arama numarasını algılamak için arama kayıtlarına ve arama durumuna erişilir. API Adresi ve Port Bilgisi: Kullanıcı tarafından manuel olarak girilir, Hızlısoft PC yazılımına veri aktarımı amacıyla saklanır. Teknik Veriler: Cihaz modeli, işletim sistemi, IP adresi gibi uygulamanın çalışması için gerekli teknik bilgiler. 2. Bilgilerin Kullanım Amacı Toplanan bilgiler şu amaçlarla kullanılır: Gelen aramaları algılayıp uygulama ekranında göstermek Arama bilgilerini API aracılığıyla Hızlısoft PC yazılımına göndermek Uygulamanın arka planda sorunsuz çalışmasını sağlamak Kullanıcı deneyimini geliştirmek ve hata ayıklamak 3. Verilerin Saklanması API adresi ve port bilgileri yalnızca cihazınızda saklanır, üçüncü taraflara aktarılmaz. Arama bilgileri sadece Hızlısoft PC yazılımına gönderilir, uygulama tarafında kalıcı olarak tutulmaz. 4. Verilerin Paylaşımı Uygulama, arama bilgilerini yalnızca kullanıcı tarafından belirlenen API adresine gönderir. Pazarlama veya reklam amacıyla hiçbir veri üçüncü kişilerle paylaşılmaz. Yasal gereklilikler haricinde (mahkeme kararı vb.) bilgiler yetkili merciler dışında kimseyle paylaşılmaz. 5. Veri Güvenliği API bilgileri ve arama verileri güvenli iletişim protokolleri (örn. HTTPS) üzerinden aktarılır. Yetkisiz erişimi önlemek için gerekli teknik ve idari önlemler alınır. 6. Kullanıcı Hakları Kullanıcılar, API bilgilerini ve uygulama erişim izinlerini diledikleri zaman silebilir veya devre dışı bırakabilir. Herhangi bir sorunda hizlisoftyazilim@gmail.com üzerinden bizimle iletişime geçebilirsiniz. 7. Politika Değişiklikleri Bu gizlilik politikası zaman zaman güncellenebilir. Güncellemeler bu sayfada yayınlandığı tarihten itibaren geçerli olur.
